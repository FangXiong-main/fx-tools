# Benchmarks

This directory contains benchmark scripts and raw data used in the SoftwareX submission.

## Directory Structure

- `json-compliance/`: JSONTestSuite compliance test for FxTools
- `coldstart-linux/`: Cold-start latency measurements on Linux (CentOS 8)
- `multi-class/`: Multi-class first-request latency evaluation

## Reproducibility

All benchmarks use Amazon Corretto 16.0.2 with `-Xmx256m`.
See individual directories for scripts and raw data.
